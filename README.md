# koyu.space Chat

koyu.space Chat interface (IRC+Jitsi). Allows people to log in with their koyu.space credentials to see both things at once. For people who don't want to use Discord, but still want to participate.

[Demo video](https://vhs.koyu.space/videos/watch/02802335-39b9-4c6c-a00f-66816e9b30b5)

## Dependencies and Installation

Most of the code is identical with the [Nordcast](https://github.com/koyuspace/nordcast) source code so the setup instructions should be the same.
